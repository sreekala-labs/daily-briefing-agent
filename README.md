# Daily Briefing Agent. 

I have created an AI Powered Morning Briefs for me to get up to date on latest topics and these are some categories I am interested in. This sends an email at 8AM PST to my email ID. 

    Tech News
    AI News,
    World News 
    Cricket News
    Finance News
    Stock Market News
    Puzzle of the Day
    Word of the Day
    Anything new in software industry
    System Design

Code Related: 

1. Using a python code
2. Using DDG to search on the web. 
3. As a SMTP Client I am using Brevo since they give 1 year of free emails to be sent out.
4. Open AI for beautifying the layout.
5. Cron Job which runs in your local laptop(Alternatives Launchd or Anacron).
